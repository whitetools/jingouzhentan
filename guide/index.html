注册流程说明

  1. 新用户可免费体验一周，当试用期结束后
  2. 点击下方注册按钮，根据提示可完成注册
  3. 注册完成后，您的手机号就是您的邀请码
  4. 用户使用你邀请码充值，会为你提供奖励

会员服务指南

  1. 当您会员到期后，可点击下方充值按钮
  2. 选择要充值的时长，输入要充值的账号
  3. 如有邀请码，输入邀请码可以享受折扣
  4. 也可以输入系统邀请码，jingouzhentan

推广奖励计划

  1. 每当好友使用您的邀请码进行充值您可以获得积分奖励
  2. 月费充值奖励1积分，年费奖励10积分，1积分=100元
  3. 可在设置中的申请提现中绑定您的支付宝或微信收款码
  4. 每满5积分可申请提现

网络配置教程

  1. 由于该软件依赖于外网环境才能使用，使用前请先进行网络配置，建议将您的机场软件更改为全局代理
  2. 更改全局后，进入功能界面后如果软件依然没办法正常加载，请点击右上角设置，进行手动配置
  3. 在软件代理这栏点击获取，然后再点击设置，如果提示获取失败，可自行填写您的代理IP和端口
  4. 一般都能在机场软件的设置中找到，如果找不到请联系您的机场软件售后进行咨询
